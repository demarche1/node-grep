# node-grep
Simple node grep clone with node streams.


## Usage

```bash
    git clone https://github.com/demarche1/node-grep.git

    cd node-grep

    npm link

    node-grep -f my-file.csv -s word
```

| argument | short | description |
| :---: | :---: | :---: |
| --file | -f | File(Required) |
| --search | -s | Search word(Optional) |
